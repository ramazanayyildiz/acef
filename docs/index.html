Ayyildiz Context Engineering Framework

ACEF

ACEF helps an AI agent understand your repo, pick the right path, and keep the work simple for the person using it. This page is the short entry point; the detailed guides stay on the other pages.

🧭What ACEF does

Understands the repo

Reads the project from evidence, not guesses.

Picks the right path

Chooses the smallest useful flow for the work.

Keeps it simple

The user says what they want. ACEF handles the rest.

💬How to start

You
I want to add a new feature to this project.
ACEF
I’ll check the repo, pick the right route, and use the project’s own patterns.
You
I need tests for this codebase.
ACEF
I’ll inspect the current setup, then choose the smallest safe test path for this repo.
